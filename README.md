💰 Personal Finance Management System (PFMS)

A modern Personal Finance Management System (PFMS) built using HTML, CSS, Bootstrap 5, and JavaScript, with data handled using MockAPI.

This project helps users track income, manage expenses, and analyze financial habits through a clean and interactive dashboard.

---

🚀 Features

🏠 Landing Page

- Clean and responsive UI
- Overview of system features
- Demo preview section
- Contact form

---

📊 Dashboard Overview

1. Financial Summary Cards:

- Total Income
- Total Expense
- Total Balance
- This Month Balance

2. 📈 Monthly Financial Trend:
   Line chart showing:

- Income (Green)
- Expense (Red)

3. 🥇 Top Categories:

- Doughnut chart for Top 3 Income Sources
- Doughnut chart for Top 3 Expense Categories

4. 📋 Recent Activity:

- Top 5 Transactions list

🛠️ Tech Stack

- Frontend: HTML, CSS, Bootstrap 5, JavaScript
- Charts: Chart.js
- Generating and manipulating pdf, table view: jsPDF, jsPDF AutoTable
- Export data to excel: SheetJS(xlsx)
- Backend (Mock): MockAPI
